# HealthSync

HealthSync streams Apple Health data to the destinations you choose with background observers and no cloud dependency. Use the per-destination guides below for exact setup steps.

## Destinations

- [AI Insights](/healthsync/ai-insights)
- [CSV Export](/healthsync/csv-export)
- [Generic Webhook](/healthsync/generic-webhook)
- [Google Sheets](/healthsync/google-sheets)
- [Grafana Cloud](/healthsync/grafana-cloud)
- [Home Assistant](/healthsync/home-assistant)
- [InfluxDB](/healthsync/influxdb)
- [Libre](/healthsync/libre) (in development)
- [n8n](/healthsync/n8n)
- [Splunk](/healthsync/splunk)

## Get started

1. Install HealthSync from the App Store.
2. Open Integrations in the app and pick a destination.
3. Follow the guide for that destination and enable more than one if you need parallel outputs.
