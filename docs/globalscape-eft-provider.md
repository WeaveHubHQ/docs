# Globalscape EFT Provider

Custom Terraform provider for Globalscape EFT resources so infrastructure changes stay versioned and repeatable.

## Setup

1. Fetch the provider from https://github.com/WeaveHubHQ/globalscape-eft-terraform-provider.
2. Configure provider credentials and endpoints in your Terraform configuration.
3. Define EFT resources in Terraform and apply to manage them through code.
