# U by Moen

Home Assistant integration for Moen water devices so you can monitor flow, usage, and valve alerts in your automations.

## Setup

1. Install the integration from the repository: https://github.com/WeaveHubHQ/ha-u-by-moen.
2. Configure your Moen account in Home Assistant.
3. Use the created sensors in dashboards and automations for water monitoring and alerts.
