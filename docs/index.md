# WeaveHub integrations

Local-first apps and integrations for Apple Health, Home Assistant, and the automations you already run. Use this docs set to pick an integration, follow its setup steps, and see what data flows where.

- HealthSync — iOS app that streams Apple Health metrics to Home Assistant, Google Sheets, Grafana Cloud, InfluxDB, CSV, and webhooks with background observers. See per-destination guides in the HealthSync page.
- Live Outage Dashboard — status and incident board built from your monitoring signals, ready to embed or publish as a status page.
- HACS Integration Preflight — VS Code extension that validates Home Assistant custom integrations for HACS with autofix commands.
- Vibe Badge — ESP32 wearable display controlled by an iOS app and Shortcuts to broadcast your energy level.
- U by Moen — Home Assistant integration for Moen water devices (flow, usage, alerts).
- Tovala — Home Assistant integration to monitor cook state and notifications from Tovala ovens.
- Robinhood Crypto — Home Assistant sensors for Robinhood crypto balances and prices.
- EFT Terraforming — Terraform workflows for Globalscape EFT to standardize provisioning.
- Globalscape EFT Provider — Terraform provider for Globalscape EFT resources with auditable changes.
- Scott's Guide — iOS reference app for bullet reloading data and procedures.

Ready to dive in? Open any integration from the left sidebar to view its dedicated guide. HealthSync includes sub-sections for every supported destination so you can jump straight to the output you plan to use.
