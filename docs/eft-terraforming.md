# EFT Terraforming

Terraform workflows for Globalscape EFT so you can standardize provisioning and manage configurations as code.

## Setup

1. Clone the workflow repository: https://github.com/WeaveHubHQ/eft-terraforming.
2. Review the modules and examples, then supply your EFT connection details and desired resources.
3. Apply via Terraform to create and update EFT resources with auditability.
